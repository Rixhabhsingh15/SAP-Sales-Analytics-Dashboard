# 📊 Sales Analytics Dashboard (SAP O2C)

## 🔹 Project Overview
This project demonstrates a Sales Analytics Dashboard developed using Power BI, based on SAP Order-to-Cash (O2C) business process. It simulates how an SAP Data/Analytics Engineer analyzes enterprise sales data to generate insights.

---

## 🔹 Objective
- To analyze sales data using modern analytics tools  
- To apply SAP O2C (Order-to-Cash) business concepts  
- To build an interactive dashboard for business decision-making  

---

## 🔹 SAP Concept Used
This project is based on SAP SD (Sales & Distribution) module and follows the O2C process:

Customer → Sales Order → Delivery → Billing → Payment  

---

## 🔹 Features
- 📈 Total Sales and Orders KPI  
- 📊 Product-wise sales analysis  
- 📅 Monthly sales trend visualization  
- 👥 Customer-wise contribution  
- 🎛️ Interactive filters for dynamic analysis  

---

## 🔹 Tools & Technologies
- Power BI Desktop  
- Microsoft Excel  
- SAP Concepts (S/4HANA, SD, O2C)
